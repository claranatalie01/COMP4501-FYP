# Data-Driven Success Roadmap for Small E-Commerce Sellers: A Predictive Analytics Approach on Shopee Indonesia Platform

**COMP4501 Final Year Project – The University of Hong Kong (HKU)**

## Objective

The primary objective of this project is to conduct a comprehensive analysis of seller performance drivers across four Shopee Indonesia categories: sandal, wallet, sunscreen, and pan.

## Data Source

Data were obtained from BrightData.

## Project Structure

- `eda/`  
  Exploratory Data Analysis (EDA) notebooks, data visualizations, and related resources.

- `machine_learning/`  
  Machine learning pipeline and modeling scripts.
